Portfolio

Root dice roll program

  • Program I made in my spare time in python which simulates the outcome of a battle in the board game Root.
  • The program rolls two 4-sided dice (with values of 0, 1, 2 and 3) and gives the higher roll to the attacker.

Kitchen converter

  • Written in python, this program can convert butter, flour, and sugar from grams to cups or vice versa.
  • Because butter, flour, and sugar have different conversion ratios from each other, different methods were needed to calculate the conversions for each ingredient.
  • Originally made as a project for an AP computer science class.
